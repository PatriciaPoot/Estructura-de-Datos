POSTRES
- ¿Qué sucedió con las listas? 
Las listas se utilizaron para almacenar los postres y los ingredientes de cada postre. Cada postre se guarda en el arreglo POSTRES y a cada uno se le asocia una lista con sus ingredientes. Además, tanto los postres como los ingredientes se organizan alfabéticamente y se verifican para evitar que se repitan.

- Conclusión:
Las listas permiten guardar y organizar varios datos relacionados dentro de un mismo programa. En este caso, facilitan almacenar los postres y sus ingredientes, modificarlos, eliminarlos y consultarlos. También permiten aplicar validaciones, como evitar duplicados y mantener el orden alfabético, lo que hace que la información sea más clara y fácil de manejar dentro del programa.
